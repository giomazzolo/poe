# Exile Tools

Path of Exile utilities published as a static site.

| Tool | Path |
| --- | --- |
| Hub | `/` |
| Exile Roulette | `/roulette/` |

## Local

```bat
serve.bat
```

Opens the hub at `http://127.0.0.1:8080/` (roulette at `/roulette/`).
